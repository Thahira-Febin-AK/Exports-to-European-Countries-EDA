# 📊 Exports to European Countries – Data Analysis Project

## 📌 Project Overview
This project analyzes export data to European countries using Python.  
The objective is to uncover patterns, trends, and key insights related to export value, quantity, countries, and commodities.

---

## 🎯 Objectives
- Understand export distribution across countries and regions
- Identify top-performing countries and commodities
- Analyze relationship between quantity and export value
- Study export trends over time
- Perform data cleaning and feature engineering

---

## 📂 Dataset Description
The dataset contains export records with the following key columns:

- Date  
- Country Name  
- Region / Sub-region  
- Commodity Name  
- Quantity of commodity  
- Export value in INR  
- Export value in USD  

---

## 🧹 Data Preprocessing
- Handled missing values
- Removed duplicate records
- Treated outliers using percentile method
- Converted date column to datetime format
- Created new features:
  - Year, Month
  - Price per unit
  - Value categories

---

## 📊 Exploratory Data Analysis (EDA)

### 🔹 Univariate Analysis

### 🔹 Bivariate Analysis
- Country vs export value

## 📈 Key Insights
- Export values are highly skewed, with few high-value transactions dominating
- A small number of countries contribute most of the export value
- Certain commodities drive the majority of revenue
- Positive relationship exists between quantity and export value
- Export trends fluctuate over time, indicating market changes
